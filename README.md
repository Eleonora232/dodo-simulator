# 🦤 Dodo Simulator

An interactive **Mathematical Ecology** playground exploring Island Biogeography and the "Island Rule" (Foster's Rule).

This project simulates species equilibrium based on:
$$S = cA^z$$
And models morphological drift (Gigantism/Dwarfism) towards an island optimum:
$$\Delta Size = \kappa (Size_{opt} - Size_{current})$$
